# 项目名称：SalaryMS
## 项目介绍：数据库课设项目，提供基本功能:系统登陆身份核验，多条件查询、新增、修改和 删除员工信息、统计月季发放总工资、分类录入数据等，帮助财务人员进行公司的员工薪资发放管理。
## 所做工作：项目负责人，搭建框架，建立数据库， UI 层及功能实现代码开发
## 亮点/技术：精美的 UI ，MVC 框架，Sqlite 3，SQL(多条件查询与多结果查询等)
## 取得成果：课设成绩排名 21/143
